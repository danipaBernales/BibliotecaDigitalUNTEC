# 📚 Biblioteca Digital UNTEC

Sistema web dinámico desarrollado con Java EE.

---

## 🚀 Tecnologías

- Java EE
- JSP
- Servlets
- JSTL
- JDBC
- MySQL
- Apache Tomcat

---

## 📂 Arquitectura

- MVC
- DAO
- JDBC Singleton

---

## ▶️ Ejecutar

1. Importar proyecto en Eclipse EE
2. Configurar Tomcat
3. Ejecutar script SQL
4. Deploy WAR
5. Abrir:

http://localhost:8080/BibliotecaDigitalUNTEC
